SET

The goal of the game is to find SETs of three cards from the twelve cards placed on the desk. All cards havefour properties:

  1. Shape
  2. Color
  3. Number
  4. Shading

Every SET consists of three cards, where all properties are:

  • the same on all of the cards
  • different on all of the cards.
